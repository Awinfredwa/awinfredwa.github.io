Got idea for a game name?

Yes, VR_BiuBiu is just the name of our project folder. It's only gameplay and no lore at this point so I'm not too sure what to call it yet. If you have a cool edgy name in mind please let me know!

New Feature: Haptics

After getting some feedback from playtests, now the controller vibrates slightly each time the player anchors to an object. It also vibrates when hitting an enemy with gun or katana.

It's a work in progress...

When gaming in VR, still moving around with a joystick just feels wrong. So I came up with this fast paced first person shooter where every move feels like a special moving ability in another game. The game is still in early development with one game mode/map of 1v1 deathmatch.

How it works is that you can anchor onto any object in the game with the pink ray by holding on to the side trigger, and either push yourself off of it or drag yourself towards it. Te rest is just shooting and slashing. Check out the video, it's a lot more fun than it sounds!

Rigging the armature

We are using Unity's IK rig along with our custom code connecting the rig to VR inputs to animate our armature Bob. Hopefully this can be easily extended to other character models in the future.

Networking

We currently are using Photon's PUN for networking. It's a great tool for quickly setting up a multiplayer game as it provides a built in server. For most of the networking logic we used PhotonView and PunRPC functions to sync game objects' statuses.

Version Control

As a two people team, we currently are using Unity's Plastic SCM for version control.

Next Steps ...

There are so many things left undone so I'll just list a few here. We plan on switching our networking from Photon to Mirror to allow for more independent control of the server. That means a whole restructuring of our networking code and a new system for lobby and matchmaking.

And of course, more variaties of weapons, more game modes, and most importantly some actual models and animations for the character and weapon. We don't have an artist on the team so if you're interested in working on this hit me up!
